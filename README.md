# RotateScreen

YouTube動画を任意の角度に回転表示するChrome拡張機能です。

横向きで撮影された動画を正しい向きで視聴できます。

## 機能

- 動画を90度、180度、270度に回転
- キーボードショートカット（Rキー）で素早く回転
- 動画プレイヤー上のボタンで操作
- 動画ごとの回転設定を記憶

## インストール方法

1. [Releases](https://github.com/ymoage/RotateScreen/releases)から最新の`RotateScreen-v1.0.0.zip`をダウンロード
2. ZIPファイルを解凍
3. Chromeで `chrome://extensions` を開く
4. 右上の「デベロッパーモード」をONにする
5. 「パッケージ化されていない拡張機能を読み込む」をクリック
6. 解凍したフォルダを選択

## 使い方

1. YouTubeで動画を開く
2. 動画プレイヤーの右上に表示される回転ボタンをクリック、または**Rキー**を押す
3. クリックするたびに90度ずつ回転します
